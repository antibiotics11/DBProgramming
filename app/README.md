## 디렉터리 구조

- <b>assets/</b>: 정적 리소스 (CSS, JS, 이미지 등)
- <b>html/</b>: 뷰(HTML) 파일
- <b>index/</b>: 웹 서버의 분산설정파일 및 색인 파일
- <b>classes/</b>: 소스 코드
- <b>vendor/</b>: 서드파티 패키지
